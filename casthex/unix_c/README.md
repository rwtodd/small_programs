# Casthex - UNIX Style

This is a UNIXy version of casthex, which:

 * Is written in _C_
 * Reads from __stdin__ when no arguments are given
 * Allows multiple args/multiple __stdin__ lines
 * Has a man page

Otherwise, it does the same job that the other versions of this
program do.
